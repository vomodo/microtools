# Text Case Converter (Microtools)

Chuyển đổi nhanh văn bản giữa các kiểu chữ: UPPERCASE, lowercase, Title Case, camelCase, snake_case.

## Các chức năng hỗ trợ
- Chữ in hoa (UPPERCASE)
- Chữ thường (lowercase)
- Title Case
- camelCase
- snake_case

Được đặt trong packages/text-case-converter/ của monorepo microtools.
