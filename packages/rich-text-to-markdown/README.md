# Rich Text to Markdown (Microtools)

**Convert rich text (HTML) to Markdown quickly. For use as standalone tool or as part of Microtools Chrome/Web suite.**

## Usage
- Integrate into extension popup, content script, or web UI.

## Files
- `converter.js` — HTML to Markdown logic
- `index.html`, `styles.css` — Demo page/UI

Put all files for this tool under `packages/rich-text-to-markdown/` in the monorepo.
