# Microtools – AI Kickstart

1. Đọc SPACE_GUIDE.md: quy tắc, flow tối giản
2. Tham chiếu TOOL_TEMPLATE.md khi tạo tool mới
3. Khi sửa/viết code, luôn import asset/css đúng chuẩn
4. Commit và PR luôn dùng template checklist chuẩn

## Quick Checklist AI/dev
- Không thêm thư viện/framework
- Code chạy full client (browser)
- CSS dùng asset/css, không copy lại
- Mỗi tool một folder, riêng biệt
- Test Chrome, Firefox, mobile
- PR ghi rõ thay đổi và checklist

Chỉ cần tuân thủ guide này, mọi refactor hoặc phát triển mới đều đảm bảo chất lượng và scale được.
